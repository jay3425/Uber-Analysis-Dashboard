# 🚖 Uber Analysis Dashboard

An interactive **Power BI Dashboard** designed to analyze Uber Bike booking performance, revenue trends, ride completion status, and operational metrics. This project provides actionable insights through visually appealing and easy-to-understand analytics.

---

## 📊 Dashboard Overview

The **Uber Analysis Dashboard** helps stakeholders monitor key business metrics, identify trends, and improve operational efficiency through data-driven decision-making.

### Key Metrics
- 🚴 **33K Total Bookings**
- 💰 **11M Total Revenue**
- 📍 **558.12K Total Distance Covered**
- 📏 **24.76 Average Distance per Ride**
- ❌ **12K Lost Bookings**

---

## ✨ Features

### Booking Analysis
- Monthly booking trends
- Total booking count tracking
- Seasonal performance monitoring

### Revenue Insights
- Revenue analysis by vehicle type
- Monthly revenue trends
- Revenue performance comparison

### Ride Status Monitoring
- Completed rides
- Cancelled rides
- Incomplete rides

### Operational Insights
- Top pickup locations
- Top drop locations
- Driver ratings analysis
- Cancelled ride tracking

### Interactive Filters
- Month-wise filtering
- Quarter-wise filtering
- Dynamic visual updates
